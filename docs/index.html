Foxit Document Generation API

Builds a Word template with Foxit token syntax, merges your release JSON, and returns a professionally formatted PDF with tables for features, fixes, and breaking changes — all in one synchronous API call.

cloud_sync

Foxit PDF Services API

Post-processes the generated PDF: adds an "INTERNAL" or "DRAFT" watermark, flattens annotations for archival, and applies password protection — all through async task-based REST endpoints.
